## ðŸ› ï¸ How to Use

1.  **Job Details**: Enter the **Job Description** (e.g., "Server Maintenance") and **Location**.
2.  **Add Shifts**:
    *   Use the `+ Add Another Day` button to add as many shifts as required.
    *   Set the **"Left House"** and **"Returned Home"** times for each.
3.  **Refine Hours**:
    *   Enter any **Break Duration** in minutes.
    *   (Optional) Enter **Driving Minutes** for travel there and back.
4.  **Toggle Breakdown**:
    *   Check `Show Driving & On-Site Breakdown` if you need to see travel time separated from work time.
5.  **Calculate**:
    *   Click **Calculate Hours** to generate your summary.
6.  **Export**:
    *   Click the **Copy** icon next to the result to grab your invoice snippet!
